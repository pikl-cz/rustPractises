TiTr | Time Tracker

- umí

  - trackovat čas strávený na různých úkolech v CLI (napříč různámi terminály)
    - vytvořit úkol
    - spustit/pozastavit/ukončit úkol
    - zobrazit přehled aktivních a ukončených úkolů (dle datumu)
  - vytvořit "minutku" pro oddychové pauzy
  - zobrazit reporty o stráveném čase

- titr je napsaný v Rust a je dostupný jako binárka pro Linux, MacOS i Windows
- data jsou ukládána lokálně v JSON formátu (žádné cloudové služby)
